What is Spring-Boot

Spring Boot is a framework that allows you to create a stand-alone, production-grade Spring based application that you can "just run".



What is Thymeleaf

Thymeleaf is a modern server-side Java template engine for both web and standalone environments. With Thymeleaf you can create HTML5 templates that are fully compatible with all modern browsers.

This application is using Thymeleaf to render the HTML pages. They can be found under the templates.
